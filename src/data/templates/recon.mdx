# Recon

Discover what the target exposes: open ports, running services,
and reachable subdomains. Two agent nodes run sequentially:

1. **`scan`** — runs the `nmap-agent` against the target.
2. **`enrich`** — feeds the scan output into `shodan-agent` for
   passive-source enrichment.

## Override before running

| Field | What to set |
|---|---|
| `target_ref` | The target's name or ID. |

## Output shape

The mission's findings table holds discovered ports, banners,
and Shodan-enriched service metadata. Use the dashboard's mission
view to inspect; the daemon also exposes findings via the
ListMissionFindings RPC.

## Customizations

- Swap `nmap-agent` for a custom scanner agent — the rest of the
  graph stays the same.
- Add a third node downstream of `enrich` to summarize findings.
