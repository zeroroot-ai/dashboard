Gibson is an AI-agent orchestration platform designed to move security R&D
to production in a single day. These docs cover the Agent Development Kit
(ADK) — the SDK, CLI, and conventions that let you vibe-code an agent and
deploy it to any runtime that can dial `api.zero-day.ai`.

This site is a stub — contributions welcome. Source lives at
[github.com/zero-day-ai/gibson-adk](https://github.com/zero-day-ai/gibson-adk).
